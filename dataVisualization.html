Data Visualization

What is Data Visualization?

Data visualization is a technique that displays data as visual diagrams, graphs, charts, or other graphics. This presents the data in an eye-catching way that, in a single glance, can be made sense of.

Tableau

The data visualization software "Tableau" gives the user an interface in which to create and view their data visualization project, as well as gives the ability to take data from spreadsheets, databases, and other data sources and import them for easy use. It contains in-depth tooling for interactive visualization, including atlas-mapping and mouse-over sectioning.

Tableau is a priced product with a free trial and free public version available. Pricing is $35/month for the personal edition and $70/month for the professional edition.
